# Flexbox

The flexbox CSS properties give you more control over your layout without using floats or positioning.

- flex-basis
- flex-direction
- flex-grow
- flex-shrink
- flex-wrap
- align-items
- align-self
- align-content
- justify-items
- justify-self
- justify-content
