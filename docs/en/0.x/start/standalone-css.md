# CSS

A quick get-started guide to using the Preons theme with plain CSS.

Download the latest css file https://unpkg.com/preons/dist/preons.css.

```bash
wget https://unpkg.com/preons/dist/preons.css
```

Add to HTML.

```bash
touch index.html
```
